# madmax-sap-order-automation
Automação desenvolvida em Excel VBA para orquestrar a captura, validação e submissão de pedidos em uma transação SAP customizada. O projeto inclui interação com ALV Grid, leitura de mensagens de erro, tratamento de exceções em tempo real e geração de trilha de auditoria, com foco em eficiência operacional e confiabilidade dos dados.
